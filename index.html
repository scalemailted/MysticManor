Mystic Manor

Haunted-house adventure in HTML + CSS + minimal JS

A minimalist, file-per-scene interactive story set in a cursed estate. Each room is a plain HTML page styled as an eerie panel. Explore the grounds, collect key items, and decide how the haunting ends.

Play Game View Code
How to play:
  • Begin at Welcome and click Proceed (this seeds the URL flags).
  • Explore the yard → Backyard; visit the Shed (machete) or the Trapdoor (basement).
  • Enter the manor and choose rooms; some choices escalate danger and push you upstairs.
  • Reach the Study to resolve the mystery—your items determine the ending.
  • State is passed in the URL (e.g., ?sk=1&mc=0&kn=0&nl=0&open=0).
Built with just HTML, CSS, and tiny bits of JavaScript. No frameworks.
